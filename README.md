# Passos para rodar aplicação

## Será necessário ter o nodejs instalado na sua máquina.

- https://nodejs.org/pt

## Clone o repositório

```
git clone https://github.com/JoaoMarcosVilela/teste2-banco.git
```

## Entre na pasta do projeto

```
cd teste2-banco
```

## Entre na pasta do backend

```
cd backend
```

## Rode a aplicação

```
node server.js
```
